Add fields and widgets for 3rd party booking services like OpenTable to your listings.

== Description ==

Allow your users to add their OpenTable ID to a listing and display the booking form through a widget.

== Installation ==

1. Install and Activate
2. Go to "Job Listings > Settings > Booking Services" and activate the services that you want.
